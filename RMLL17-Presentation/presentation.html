Why does learning-by-doing suffer?

  • Experiments are often given a lesser role in Science Education

  • This may be due to a variety of reasons

    • Lack of cost effective laboratory equipment

    • Absence of funds

    • issues with availability

    • Lack of technical expertise to leverage existing technology

    • A well equipped laboratory can cost thousand of euros in capital investment and maintenance charges. Most instruments require significant electrical resources

------------------------------------

What can be done to help solve this problem?

  • Utilize existing technology

  • Create a modular framework that can be improved at minimal cost

  • Leverage open source frameworks

  • translation-friendly, plain-text documentation

  • A common knowledge sharing platform.

For example, modern day phones and computers have exceedingly good visualisation and analytical resources, and with some added tools to access the real world, these can be made into platforms for exploration.

What is ExpEYES, and how can it help?

  • Multi-purpose, USB powered data acquisition tool

  • Highly extensible framework accessible via Python library

  • Affordable. Minimal Bill of Materials

  • translation-friendly documentation

  • Does not depend on proprietary tools

  • Fully open-source

Extend Functionality using common sensors

  • Eliminate the need for expertise in electronics.
  • Shift focus to gathering data and performing experiments
Application : A Pendulum studied using an accelerometer
  • Angle is measured using the direction of the acceleration vector
    and the known direction of acceleration due to gravity.
  • Speed of Oscillations is measured with a gyroscope.

Some Supported Sensors

  • MPU6050 : 3-Axis Accelerometer , 3-Axis GyroScope
  • HMC5883L : 3-Axis Magnetometer with adjustable ranges
  • MLX90614 : Passive IR temperature sensor
  • BMP180 : Pressure , Temperature and Altitude module.
  • TSL2561 , BH1750 : Luminosity measurements up to 40K Lux
  • SHT21 , Si7021 : Ambient Temperature and Humidity Module
  • AD9833 : 28-bit DDS function generator
  • PT100 temperature sensor, MQ Series Gas sensors
  • HX711 24-bit differential ADC with 128x PGA

Many more can be added with a spot of Python code!

ExpEYES Project : a brief history

  • 2005: The PHOENIX (Physics with Home-made Equipment and Innovative Experiments) project, started by IUAC to address this issue.
  • 2006: Next version: Powered by 9V DC adapter. 6” x 5” x 1” size. Code written in Python programming language
  • 2011: ExpEYES : GUIs for 50 experiments , voltmeters, oscilloscopes, Python Powered.
  • 2012: ExpEYES-junior : Smaller, faster, and more cost-effective

ExpEYES Project : a brief history

  • The brainchild of Dr. Ajith Kumar from the Inter University Accelerator Center, New Delhi.
  • Contributions by V.V.V. Satyanarayana , Pramode C.E , towards experiment development and outreach programmes.
  • Contributions by Praveen Patil and Akshai M for improving various aspects and popularising this project at several conferences
  • French translations ,software development, and Debian packaging by Monsieur Georges Khaznadar
  • Development of a Pocket Science Lab inspired from ExpEYES has also been taken up by FOSSASIA . This project has developers working with a focus on Android and Mobile accessibility
