# ExpEYES17-Qt
Qt based toolkit for accessing ExpEYES-17

Dependencies
============
python-pyqt5 , python-pyqtgraph, python-scipy, python-pyqt5.qtsvg
pyqt5-dev-tools


Building
========
cd SPARK17
make

Housekeeping
============
`make clean` wipes out all generated files and leves only sources.
